# 勝敗判定機能の実装記録

## 実装日時
2025年8月2日

## 作業時間
約15分

## ブランチ
feature/develop/#q4_result

## 実装概要
ゲームが終了したら（どちらも石を置けなくなったら）、勝敗を判定して結果を表示する機能を追加します。

## 実装内容
1. ゲーム状態管理の追加
   - `_isGameOver`フラグを追加
   - 結果表示用の`_resultText`参照を追加

2. ゲーム終了判定の実装
   - 現在のプレイヤーが置けない場合、相手も置けるかチェック
   - どちらも置けない場合にゲーム終了とする

3. 勝敗判定と結果表示の実装
   - 石の数を数えて勝敗を判定
   - "ゲーム終了"と勝者を表示
   - 最終的な石の数も表示

4. パスの処理
   - 置ける場所がない場合は自動的にパス
   - パス後に相手も置けない場合はゲーム終了

## 表示内容
- 開始時: 結果テキストは非表示
- 終了時のみ表示:
  - 勝利時: "あなたの勝ち！"
  - 敗北時: "AIの勝ち！"
  - 引分時: "引き分け"
  - 各結果に加えて "ゲーム終了" と最終的な石の数も表示

## テスト項目
- [ ] 置ける場所がなくなった時に正しくパスされるか
- [ ] どちらも置けなくなった時に正しくゲームが終了するか
- [ ] 勝敗判定が正しく行われているか
- [ ] 結果テキストが正しく表示されるか
- [ ] ゲーム終了後に操作を受け付けなくなるか

## 必要な作業
1. UIにTextMeshProUGUIコンポーネントを配置（結果表示用）
2. GameControllerの`_resultText`にコンポーネントを設定
